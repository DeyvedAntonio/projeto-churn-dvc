# Título

Modelo de previsão de Churn

## Descrição do problema

Criar um modelo de Machine Learning para prever Churn.

### Contexto do Cenário

Sou Engenheiro de Machine Learning recém-contratado por um banco digital.
Minha primeira tarefa é criar um modelo para prever Churn.
O Tech Lead impôs uma regra importante: nada  de  notebooks isolados. 
O  projeto  deve  ser  reprodutível,  auditável  e  seguir  as  melhores práticas da indústria.

## Pré-requisitos

1. Python;
2. Git;
3. DVC;
4. pandas;
5. scikit-learn;
6. pyyaml

## Instalação e Setup

### Como executar:

1. Clone o repositório
```
git clone https://github.com/DeyvedAntonio/projeto-churn-dvc
```

2. Instale as dependências

```
pip install -r requirements.txt
```

3. Puxe os dados e modelos (DVC)

```
dvc pull
```

4. Reproduza o pipeline

```
dvc repro
```
